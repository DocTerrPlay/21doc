1. Общее

   Блэк Джек (21) - популярная карточная игра, которая может стать отличным способом с интересом провести
     время в ожидании учителя или конца перемены
   Цель проекта: воссоздать копию игру со всеми правилами и загрузить на свой сайт

2. Цели

   2.1 Создать свой сайт и загрузить его на GitHub (15.01)
   2.2 Найти правила игры 21 и необходимые изображения карт (03.02)
        2.2.1 Создать кнопку "?", при нажатии которой будут отображаться текст правил игры 21 
   2.3 Оформить сайт с помощью .html и .css файлов (20.01)
        2.3.1 Придумать каким сделать задний фон и стиль текстов
        2.3.2 Создать Кнопки "Взять" и "Хватит" чуть ниже центра экрана
        2.3.3 Также в центре отображать все использующиеся карты игрока, карты Дилера и колоду, из которой происходит добор карт кнопкой "Взять" и Дилером
   2.4 Составить код на JavaScript и с помощью туториалов изучить необходимые знания языка (10.02) 
        2.4.1 Написать код для кнопок "Взять" и "Хватит", чтобы соответственно брать новые карты в руку или передать ход Дилеру
        2.4.2 Написать код для Победы, Поражения и Ничьи (условия их возникновения)
        2.4.3 Написать код для Дилера (в каких случаях от берёт карты)
        2.4.4 Написать код для Колоды карт так, чтобы при взятии определённой карты она исчезала из колоды и появлялась на столе
   2.5 Сдать рабочий проект учителю (24.02)

3. Приложения

   3.1 Блокнот для .html и .css файлов (установлен на компьютере)
   3.2 GitHub для размещения сайта (онлайн сайт)
   3.3 Visual Studio Code для написания .js файла (установлен на моём домашнем компьютере)
